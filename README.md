# epubReader

A simple, aesthetically pleasing EPUB and TXT file reader for reading novels, built with plain HTML, CSS, and JavaScript.

## Features

- 📂 **File input** – click the drop zone or drag-and-drop a `.epub` or `.txt` file
- 📖 **EPUB rendering** – powered by [epub.js](https://github.com/futurepress/epub.js); preserves chapters, fonts, and images
- 📄 **TXT rendering** – plain-text files are displayed with comfortable typography
- 🌙 **Dark / Light mode** – toggle with the moon/sun button in the toolbar
- 🔠 **Font-size controls** – increase or decrease text size with `A+` / `A−`
- ⌨️ **Keyboard navigation** – use ← → or ↑ ↓ arrow keys to turn EPUB pages

## Usage

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
2. Drop a `.epub` or `.txt` file onto the drop zone, or click it to browse.
3. Use the toolbar controls to adjust font size or toggle the theme.
4. Click **←** (back arrow) to load a different file.

> **No installation or server required** – everything runs locally in the browser.
